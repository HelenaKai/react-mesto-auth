# Проект: Mesto React с авторизацией (react-mesto-auth)

### Версии проекта
Проектная работа по созданию интерактивного веб-сервиса на REACT, созданная в рамках прохождения 12 спринта курса Веб-разработчик в Яндекс.Практикум.

### Стек технологий:
* HTML 5
* CSS 3
* БЭМ Nested
* JavaScript
* React
* API;
* Fetch;
* Promise;
* React;
* React Router.
* Git
